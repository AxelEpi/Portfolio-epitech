## Contexte

Projet Epitech — création d'une plateforme d'offres d'emploi complète avec frontend, backend et base de données.

## Fonctionnalités

- Affichage des offres d'emploi
- Détail d'une offre sans rechargement de page
- Formulaire de candidature
- Authentification utilisateur
- Espace admin avec CRUD complet
- Pagination des listes

## Stack technique

| Composant | Technologie |
|-----------|-------------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Node.js, Express |
| Base de données | SQL |
| Auth | Sessions |

## Étapes du projet

1. Création de la base de données SQL
2. Page HTML/CSS des offres
3. Affichage dynamique sans rechargement
4. API REST avec opérations CRUD
5. Formulaire de candidature
6. Authentification
7. Interface d'administration
8. Polish et design final
