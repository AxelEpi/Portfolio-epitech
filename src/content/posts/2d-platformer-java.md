## Concept

Jeu de plateforme 2D développé en Java. Les joueurs doivent récupérer une boîte sur la droite de la map et la déplacer jusqu'à l'emplacement cible en haut à gauche.

## Fonctionnalités

- Gameplay de type puzzle-plateforme
- Compétences joueurs utilisables
- 2 skins au choix
- Carte avec zones distinctes

## Stack technique

- Java
- Développement avec VS Code
